20 Web Development Terms

HTML
HyperText Markup Language, the standard language for creating web pages.
CSS
Cascading Style Sheets, used to style and layout web pages.
JavaScript
A programming language that adds interactivity and dynamic behavior to websites.
DOM
Document Object Model, a programming interface that represents the structure of a webpage.
API
Application Programming Interface, a set of rules that allows software components to communicate.
Frontend
The part of a website users interact with directly (HTML, CSS, JavaScript).
Backend
The server-side logic, databases, and application code that power a website.
HTTP
HyperText Transfer Protocol, the foundation of data communication on the web.
HTTPS
Secure version of HTTP, using encryption (SSL/TLS) for secure communication.
URL
Uniform Resource Locator, the address used to access resources on the web.
Responsive Design
A design approach ensuring websites work on all devices and screen sizes.
SEO
Search Engine Optimization, techniques to improve website ranking on search engines.
Accessibility
Practices that ensure web content is usable by people with disabilities.
Framework
A pre-built collection of tools and libraries that speeds up development (e.g., React, Angular).
Library
A collection of reusable code that developers can call in their applications (e.g., jQuery).
Git
A version control system for tracking changes in code.
Repository
A storage location for project files, often hosted on platforms like GitHub.
Database
An organized collection of data, often managed by a system like MySQL or MongoDB.
Server
A computer or system that provides resources, data, or services to clients over a network.
Client
The device or application that requests and uses resources from a server, usually a browser.
